# BSPHP — dylib.verify.oc（iOS 动态库 + VerifyHost）

## 项目简介

iOS 测试工程：`VerifyHost.app` 启动后 `dlopen` 包内 `verify.dylib`，便于在 Xcode 中调试 constructor、网络、UIKit 弹窗。细节见同目录 **说明.md**。

## 目录结构

```
dylib.verify.oc/
├── verify.xcodeproj/          Scheme：verify、VerifyHost
├── verify/
│   ├── entry.mm
│   ├── NetTool/               Config.h 等
│   ├── AFNetworking/
│   └── Category/
├── VerifyHost/
├── 编译好的测试/
├── 配置说明/
├── 说明.md
└── 说明中文.md / 说明繁体.md / 说明英文.md
```

## 配置说明

编辑 **`verify/NetTool/Config.h`**：`BSPHP_HOST`、`BSPHP_MUTUALKEY` 及 RSA 相关宏。

## 调试（摘要）

1. 打开 `verify.xcodeproj`  
2. Scheme：**VerifyHost**，目标：iOS 模拟器或真机  
3. ⌘R；控制台查看 `dlopen` 与验证日志  

只编 dylib：Scheme 选 **verify**。
