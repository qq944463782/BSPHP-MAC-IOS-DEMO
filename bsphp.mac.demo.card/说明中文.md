# BSPHP — bsphp.mac.demo.card（macOS 卡模式演示 App）

## 项目简介

在 Mac 上演示 BSPHP 卡密 / 机器码登录与控制台（SwiftUI）。与 **bsphp.mac.demo**（账号模式）互补，工程独立。

## 目录结构

```
bsphp.mac.demo.card/
├── bsphp.mac.demo.card.xcodeproj/
├── bsphp.mac.demo.card/
│   ├── bsphp_mac_demo_cardApp.swift
│   ├── ContentView.swift              BSPHPCardConfig
│   ├── CardMainControlPanelView.swift BSPHPCardSaleWeb（daihao 等）
│   ├── bsphp_api_http/
│   ├── Assets.xcassets/
│   └── Preview Content/
├── bsphp.mac.demo.cardTests/
├── bsphp.mac.demo.cardUITests/
├── 效果图-配置说明/
├── 说明中文.md / 说明繁体.md / 说明英文.md
└── （构建产物在 DerivedData）
```

## 配置说明

1. `ContentView.swift` → **BSPHPCardConfig**。
2. `CardMainControlPanelView.swift` → **BSPHPCardSaleWeb** 与后台软件代号一致。

## 运行

Xcode 打开工程，**My Mac**。**Product → Show Build Folder in Finder**。
