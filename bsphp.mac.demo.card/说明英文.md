# BSPHP — bsphp.mac.demo.card (macOS card-mode demo app)

## Overview

SwiftUI macOS sample for BSPHP card / machine-code login and control panel. Complements **bsphp.mac.demo** (account mode); separate project.

## Directory tree

```
bsphp.mac.demo.card/
├── bsphp.mac.demo.card.xcodeproj/
├── bsphp.mac.demo.card/
│   ├── bsphp_mac_demo_cardApp.swift
│   ├── ContentView.swift              BSPHPCardConfig
│   ├── CardMainControlPanelView.swift BSPHPCardSaleWeb
│   ├── bsphp_api_http/
│   ├── Assets.xcassets/
│   └── Preview Content/
├── bsphp.mac.demo.cardTests/
├── bsphp.mac.demo.cardUITests/
├── 效果图-配置说明/
├── 说明中文.md / 说明繁体.md / 说明英文.md
└── (Build output in DerivedData)
```

## Configuration

1. **BSPHPCardConfig** in `ContentView.swift`.
2. **BSPHPCardSaleWeb** in `CardMainControlPanelView.swift` — match backend software code.

## Run

Destination: **My Mac**. **Product → Show Build Folder in Finder**.
