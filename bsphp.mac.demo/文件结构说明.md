## 模块职责摘要

| 路径 | 说明 |
|------|------|
| `bsphp_api_http/BSPHPClient.swift` | 与 BSPHP `AppEn.php` 通讯：会话 `bootstrap`、登录注册、卡模式 `.ic`、公共 `.in` 等。 |
| `bsphp_api_http/BSPHPCrypto.swift` | 与 PHP 端一致的加解密与签名流程。 |
| `ContentView.swift` | 组装主导航，嵌入各 `demo_bsphp_*` 子视图。 |
| `demo_bsphp_*_view.swift` | 按功能拆分的 SwiftUI 页面，便于单独维护与预览。 |
| `demo_bsphp_console_window_view.swift` | 调试/日志类独立窗口示例。 |

---

## 与「卡模式」独立演示工程的关系

同仓库中的 **`bsphp.mac.demo.card`**（在上一级 `BSPHP-MAC-演示案例/` 下）是 **卡密 / 机器码** 专用精简 Demo（`login.ic`、`AddCardFeatures.key.ic`、`chong.ic` 等），与本工程 **账号模式** 演示互补；二者可共用相似的 `bsphp_api_http` 实现思路，但工程相互独立。
