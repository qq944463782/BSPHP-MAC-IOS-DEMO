# BSPHP — dylib.ios.card（iOS 卡密驗證 dylib，Swift）

## 專案簡介

iOS 動態庫：BSPHP 卡密驗證（Swift，`bsphp_api_http`）。可整合至自有 App 或透過 **VerifyHost** 以 `dlopen` 偵錯。

## 目錄結構

```
dylib.ios.card/
├── dylib.ios.card.xcodeproj/  Scheme：dylib.ios.card、VerifyHost
├── dylib.ios.card/
│   ├── IOSCardConfig.swift    連線參數
│   ├── IOSCardVerifyEntry.swift
│   ├── bsphp_api_http/
│   └── dylib_ios_card_initializer.m
├── VerifyHost/
├── 编译结果演示/
├── 配置说明/
└── 说明中文.md / 说明繁体.md / 说明英文.md
```

## 設定說明

編輯 **`dylib.ios.card/IOSCardConfig.swift`**（與後台同一應用的位址、mutualKey、RSA 等一致）。

## 偵錯（摘要）

1. 開啟 `dylib.ios.card.xcodeproj`  
2. Scheme：**VerifyHost**，目標：iOS 模擬器或真機  
3. ⌘R
