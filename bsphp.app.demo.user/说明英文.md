# BSPHP — bsphp.app.demo.user (iOS account-login demo app)

## Overview

SwiftUI iOS sample for BSPHP account register/login, captcha, recharge, etc. URL, mutualKey, RSA keys, and captcha prefix must match the same backend app.

## Directory tree

```
bsphp.app.demo.user/
├── bsphp.app.demo.user.xcodeproj/
├── bsphp.app.demo.user/
│   ├── bsphp_app_demo_userApp.swift
│   ├── ContentView.swift              BSPHPMobileConfig
│   ├── bsphp_api_http/
│   ├── Assets.xcassets/
│   └── Preview Content/
├── bsphp.app.demo.userTests/
├── bsphp.app.demo.userUITests/
├── 效果图-配置说明/
├── 说明中文.md / 说明繁体.md / 说明英文.md
└── (Build output in DerivedData)
```

## Key points

- **BSPHPMobileConfig** in **ContentView.swift**: `url`, `mutualKey`, RSA, `codeURLPrefix`.
- **bsphp_api_http/**: networking and crypto.

## Configuration

Edit **BSPHPMobileConfig** in `bsphp.app.demo.user/ContentView.swift` to match AppEn, mutual key, RSA pair, and captcha image base URL.
