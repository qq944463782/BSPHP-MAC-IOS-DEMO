# BSPHP — bsphp.app.demo.user（iOS 帳號登入模式演示 App）

## 專案簡介

演示 BSPHP 帳號註冊、登入、驗證碼、充值等能力的 iOS 應用（SwiftUI）。伺服器位址、mutualKey、RSA 及驗證碼前綴須與後台同一應用一致。

## 目錄結構

```
bsphp.app.demo.user/
├── bsphp.app.demo.user.xcodeproj/
├── bsphp.app.demo.user/
│   ├── bsphp_app_demo_userApp.swift
│   ├── ContentView.swift              BSPHPMobileConfig（API 設定）
│   ├── bsphp_api_http/
│   ├── Assets.xcassets/
│   └── Preview Content/
├── bsphp.app.demo.userTests/
├── bsphp.app.demo.userUITests/
├── 效果图-配置说明/
├── 说明中文.md / 说明繁体.md / 说明英文.md
└── （建置產物在 DerivedData）
```

## 主要說明

- **ContentView.swift** 內 **BSPHPMobileConfig**：`url`、`mutualKey`、`RSA`、`codeURLPrefix`。
- **bsphp_api_http/**：網路與加密。

## 設定說明

編輯 `bsphp.app.demo.user/ContentView.swift` 中 **BSPHPMobileConfig** 的常數，與 BSPHP 後台「當前應用」的 AppEn、通訊 KEY、RSA 金鑰對、驗證碼圖片位址前綴一致。
