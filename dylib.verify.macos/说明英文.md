# BSPHP — dylib.verify.macos (macOS dylib + VerifyHost)

## Overview

macOS BSPHP card-license dylib (same protocol as `dylib.verify.oc` / card demos). Use for `dlopen` testing or injection. See **说明.md**.

## Directory tree

```
dylib.verify.macos/
├── verify.macos.xcodeproj/    Schemes: verify.macos, VerifyHost
├── verify.macos/
│   ├── entry.m
│   └── NetTool/               Config.h, …
├── VerifyHost/
├── 编译的结果/
├── 配置说明/
├── 说明.md
└── 说明中文.md / 说明繁体.md / 说明英文.md
```

## Configuration

Edit **`verify.macos/NetTool/Config.h`**: `BSPHP_HOST`, `BSPHP_MUTUALKEY`, RSA macros.

## Usage

- Debug UI: scheme **VerifyHost**, destination **My Mac**, ⌘R.  
- Dylib only: scheme **verify.macos** → **Product → Show Build Folder in Finder**.
