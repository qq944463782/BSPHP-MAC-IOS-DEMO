# BSPHP — dylib.verify.macos（macOS 动态库 + VerifyHost）

## 项目简介

macOS 动态库：BSPHP 卡密协议（与 `dylib.verify.oc` / 卡模式 Demo 一致），可用于 `dlopen` 测试或自行注入宿主。细节见同目录 **说明.md**。

## 目录结构

```
dylib.verify.macos/
├── verify.macos.xcodeproj/    Scheme：verify.macos、VerifyHost
├── verify.macos/
│   ├── entry.m                constructor、NSAlert 流程
│   └── NetTool/               Config.h、BSPHP 客户端等
├── VerifyHost/                dlopen 测试宿主
├── 编译的结果/                示例产物（可选）
├── 配置说明/
├── 说明.md
└── 说明中文.md / 说明繁体.md / 说明英文.md
```

## 配置说明

编辑 **`verify.macos/NetTool/Config.h`**：`BSPHP_HOST`、`BSPHP_MUTUALKEY`、RSA 相关宏。

## 使用（摘要）

- 调试弹窗：Scheme **VerifyHost**，目的地 **My Mac**，⌘R。  
- 只编 dylib：Scheme **verify.macos**，**Product → Show Build Folder in Finder**。
