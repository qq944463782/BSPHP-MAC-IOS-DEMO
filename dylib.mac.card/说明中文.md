# BSPHP — dylib.mac.card（macOS 卡密验证 dylib，Swift）

## 项目简介

macOS 动态库：BSPHP 卡密验证（Swift，`bsphp_api_http` 负责网络与加解密）。加载后由 constructor 排队，待 AppKit 就绪后走公告（gg.in）与卡密（login.ic）等流程。细节见 **README.md**。

## 目录结构

```
dylib.mac.card/
├── dylib.mac.card.xcodeproj/  Scheme：dylib.mac.card、VerifyHost
├── dylib.mac.card/            动态库源码（含 MacCardVerifyEntry、dylib_mac_card.m）
│   └── bsphp_api_http/
├── VerifyHost/                dlopen 调试宿主
├── 编译结果演示/
├── 配置说明/
├── README.md
└── 说明中文.md / 说明繁体.md / 说明英文.md
```

## 产物名称

动态库：**`libdylib.mac.card.dylib`**（工程内 `EXECUTABLE_PREFIX = lib`）。

## 配置说明

编辑 **`dylib.mac.card/MacCardVerifyEntry.swift`** 中私有枚举 **`MacCardBSPHPConfig`**（`host`、`mutualKey`、`serverPrivateKey`、`clientPublicKey`）。

## 调试（摘要）

1. 打开 `dylib.mac.card.xcodeproj`  
2. Scheme：**VerifyHost**，目的地：**My Mac**  
3. ⌘R
