# BSPHP — dylib.mac.card (macOS card-verify dylib, Swift)

## Overview

macOS dylib for BSPHP card verification (Swift; `bsphp_api_http` for HTTP/crypto). Constructor queues work until AppKit is ready, then runs notice + card login flow. See **README.md**.

## Directory tree

```

│   └── bsphp_api_http/
├── VerifyHost/
├── 编译结果演示/
├── 配置说明/
├── README.md
```

## Product name

**`libdylib.mac.card.dylib`** (`EXECUTABLE_PREFIX = lib`).

## Configuration

Edit private enum **`MacCardBSPHPConfig`** in **`dylib.mac.card/MacCardVerifyEntry.swift`**.

## Debug

1. Open `dylib.mac.card.xcodeproj`  
2. Scheme **VerifyHost**, destination **My Mac**  
3. ⌘R
